## How to capture

1. Run `npm run dev` (or `docker compose up`)
2. Open **http://localhost:3000** in Chrome
3. Use **DevTools → Toggle device toolbar** (Ctrl+Shift+M) for the mobile screenshot
4. Save screenshots to this folder and reference them in the root README if needed
